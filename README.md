# Task Queue

这是我自己写的一个 Windows 桌面待办小工具。

最开始只是想要一个放在桌面上随手记任务的东西，要求很简单:

- 能快速把最新任务追加到末尾
- 能随时删掉任意一项，表示已经做完
- 能给任务标 0 到 3 星的重要程度
- 界面尽量简洁一点，放桌面上看着别太乱

后来又陆续加了一些我自己会用到的功能，比如导入导出、提醒、背景图、排序、开机启动这些。

## 现在有的功能

- 输入任务后按 `Enter` 或点击按钮可以直接追加
- 任务支持 `0-3` 星优先级
- 选中任务后可以完成、升星、降星
- 任务清单可以导出成 `txt`，也可以再导入回来
- 导入是追加，不会把原来的任务删掉
- 可以给任务设置提醒时间
- 提醒时间现在是弹窗选择，能选年月日、小时、分钟
- 支持窗口置顶
- 支持开机启动
- 支持设置背景图
- 任务和设置会自动保存，下次打开还在

## 运行环境

这个项目目前就是给 Windows 用的，建议环境:

- Windows 10 / Windows 11
- 64 位系统

正常情况下，直接运行 `build-ninja/task_queue.exe` 就可以。

我这边构建时已经把 MinGW 运行库静态链接进去了，所以把 exe 单独拷到别的 Windows 电脑上一般也能直接跑，不需要额外装 `libgcc` 之类的 dll。

## 数据保存位置

程序默认会把数据写到:

- `%APPDATA%\\TaskQueue\\tasks.txt`
- `%APPDATA%\\TaskQueue\\settings.txt`

如果系统环境里拿不到 `APPDATA`，就退回到 exe 当前目录。

## 导入导出说明

导出出来的是普通 `txt` 文件，里面会保留这些信息:

- 任务内容
- 星级
- 添加时间
- 提醒时间

所以如果只是想把自己的任务发给别人，直接导出这个文件再发过去就行。

## 提醒功能说明

提醒是本地提醒，程序运行时会检查时间，到点或者快到点时弹窗提示。

目前支持:

- 准时提醒
- 提前 5 分钟
- 提前 10 分钟
- 提前 15 分钟
- 提前 30 分钟

## 代码结构

项目结构比较直接，没有拆很复杂:

- `src/main.cpp` 负责程序入口和基础初始化
- `src/app.cpp` 主要是窗口、任务逻辑、绘制、导入导出、提醒这些
- `include/app.h` 放主类声明
- `resources/` 放程序资源，比如清单文件和图标

## 本地构建

我现在用的是 `CMake`。

```bash
cmake -S . -B build-ninja -G Ninja
cmake --build build-ninja
```

如果是在 Windows 下配好 MinGW / Ninja，编完之后可执行文件就在:

```text
build-ninja/task_queue.exe
```
